Show Country

Optional self-hosted geo lookup (e.g. cloud66-oss/geo behind your own URL). The extension calls GET {base URL}/{resolved IP} and expects JSON with either country/countryCode or a country.iso_code object (see README).

No trailing slash. Examples: http://geoip.tma → request http://geoip.tma/8.8.8.8; with a path prefix use http://geoip.tma/v1/ip → …/v1/ip/8.8.8.8.
