Show Country

Optional self-hosted geo lookup (e.g. cloud66-oss/geo behind your own URL). When a custom base URL is saved, the extension uses only GET {base URL}/{resolved IP} for country (no public providers). Clear that field to use the public providers you enable below. JSON: either country/countryCode or a country.iso_code object (see README).

No trailing slash. Examples: http://geoip.tma → request http://geoip.tma/8.8.8.8; with a path prefix use http://geoip.tma/v1/ip → …/v1/ip/8.8.8.8.

Public geo providers

Used only when the custom URL above is empty. Order is fixed; the extension tries each checked provider until one succeeds. At least one must be checked.
